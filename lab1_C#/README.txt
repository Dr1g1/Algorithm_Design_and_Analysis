KMP folder:
Implementation of the Knuth-Morris-Pratt (KMP) string matching algorithm in C#. The program searches for pattern occurrences in text files of varying sizes (100 to 100,000 words) and measures execution time performance. Supports both ASCII and hexadecimal text processing with pattern lengths ranging from 5 to 50 characters.

LevenshteinDistance folder:
Implementation of the Levenshtein Distance algorithm in C# for approximate string matching. The program finds similar patterns in text allowing for minor differences (insertions, deletions, substitutions), tested on files of varying sizes (100 to 100,000 words) with performance measurement.
